# 미세먼지 앱

### 프로젝트 소개
앱 실행시 내 위치 정보를 가져와 미세먼지 정보를 보여준다. (위젯 기능 포함)

![](https://velog.velcdn.com/images/jieun0915/post/e86904a9-98d1-4b8e-91a6-cc2922dc8cf3/image.jpg)

---

### 주요 기능
- 내 위치 정보 가져오기
- 오픈 API로부터 미세먼지 정보 가져오기
- 홈 스크린에 미세먼지 위젯 추가

---

### 사용 기술
- LocationManager
- Retrofit2
- Coroutine
- App Widgets
